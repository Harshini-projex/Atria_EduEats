
# 🎓 Atria EduEats

Atria EduEats is an all-in-one campus management platform developed for **Atria Institute of Technology**. The platform simplifies students' daily activities by integrating canteen food ordering, academic resources, campus events, and communication into a single web application.

---

## 📖 About the Project

Students often use different platforms for ordering food, accessing study materials, checking announcements, and participating in campus activities. Atria EduEats brings these essential services together to provide a seamless and user-friendly experience.

The project aims to improve campus convenience, reduce waiting times at the canteen, and make academic resources easily accessible to students.

## 🚀 Installation

### Clone the repository

```bash
git clone https://github.com/your-username/Atria-EduEats.git
```

### Navigate to the project

```bash
cd Atria-EduEats
```

### Install dependencies

```bash
npm install
```

### Start the development server

```bash
npm start
```
